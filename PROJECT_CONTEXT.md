# Portfolio 2026 — Project Context

## Overview
This is an interactive cybersecurity portfolio built using React, focused on showcasing real-world security engineering, SOC operations, and automation capabilities.

The goal is to present technical depth through UI/UX, not just static content.

---

## Tech Stack
- React (CRA)
- Tailwind CSS
- Framer Motion (animations)
- Vercel (deployment)

---

## Live URL
https://portfolio2026-blush-two.vercel.app/

---

## Core Features
- Interactive SOC-style terminal (mock telemetry + commands)
- Metrics dashboard with modal deep dive
- Glassmorphism UI with animated components
- Security-focused storytelling approach

---

## Key Sections
- Hero (intro + terminal UI)
- Metrics (impact + modal details)
- Capabilities (planned upgrade)
- Projects (planned upgrade)
