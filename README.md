# portfolio2026

Cybersecurity portfolio built with Create React App, Tailwind CSS, Framer Motion, and Lucide icons.

## Run

```bash
npm install
npm start

---

## Install the needed packages

Run this in your project root:

```bash
npm install framer-motion lucide-react
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

Then paste the files above, start with:

npm start
